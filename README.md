# Dental Groove

Professional Dental Clinic Management Software for Golden Groove Dental Clinic.

## Features

- **Appointment Management**: Book, approve, and track patient appointments.
- **Patient Records**: Securely store patient history and details.
- **Role-Based Access**: Admin and Staff dashboards.
- **Responsive Design**: Works seamlessly on desktop and mobile.

## Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS + Shadcn UI
- Supabase (Auth, Database, Realtime)
- React Query

## Getting Started

1.  Clone the repository
2.  Install dependencies: `npm install`
3.  Start development server: `npm run dev`
